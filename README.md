# Tab-Switch-Detection for interview
<h2>About the Project</h2> 

This is a simple browser-based interview monitoring system which detects when a candidate sswitches the tab, exits the full screen mode, or leaves interview window .The system keeps track of these activities and dislay them in activity log.

<h2>Technologies Used</h2>
-HTML<br>
-CSS<br>
-JavaScript<br>


<h2>Features</h2>
* Full Screen Mode<br>  
* Tab swith Detection <br>  
* Warning Counter  <br>
* Activity Log <br> 
* Timer of 1 Hour <br> 
* Automatic termination after many warnings <br>


<h2>How It Works</h2>
1. When the interview starts, the timer begins and the page enters fullscreen mode.<br>
2. If the user switches tabs, a warning is generated.<br>
3. If the user exits fullscreen mode, another warning is generated.<br>
4. All activities are recorded in the log section.<br>
5. After 3 warnings, the interview is terminated.<br>
