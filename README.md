# Tab-Switch-Detection for interview
<h1>About the Project</h1> 

This is a simple browser-based interview monitoring system which detects when a candidate sswitches the tab, exits the full screen mode, or leaves interview window .The system keeps track of these activities and dislay them in activity log.

<h1>Technologies Used</h1>
-HTML<br>
-CSS<br>
-JavaScript<br>


<h1>Features</h1>
* Full Screen Mode<br>  
* Tab swith Detection <br>  
* Warning Counter  <br>
* Activity Log <br> 
* Timer of 1 Hour <br> 
* Automatic termination after many warnings <br>


<h1>How It Works</h1>
1. When the interview starts, the timer begins and the page enters fullscreen mode.
2. If the user switches tabs, a warning is generated.
3. If the user exits fullscreen mode, another warning is generated.
4. All activities are recorded in the log section.
5. After 3 warnings, the interview is terminated.
